# PyQuick - Python安装与管理工具

PyQuick是一个用于下载、安装和管理Python及其包的图形化工具。它简化了Python环境的管理，特别适合中国用户使用镜像站点加速下载和管理Python包。

## 功能特点

- **Python下载管理**：支持多版本Python的下载和安装
- **多线程下载**：支持多线程加速下载Python安装包
- **Pip包管理**：轻松安装、卸载和更新Python包
- **镜像测试**：测试国内各大镜像站点的延迟和可用性
- **自定义镜像**：支持添加和管理自定义下载镜像
- **多语言支持**：支持中文和英文界面
- **主题切换**：支持浅色和深色主题

## 系统要求

- Windows操作系统（Windows 10及以上推荐）
- Python 3.6+（如需运行源码）

## 使用方法

### 方式一：运行可执行文件

1. 下载最新版本的PyQuick.exe
2. 双击运行即可

### 方式二：运行源码

1. 克隆本仓库或下载源码
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python run.py` 或 `python pyquick.py`

## 主要功能使用说明

### Python下载

1. 在「Python下载」选项卡中选择Python版本
2. 选择安装文件和目标保存路径
3. 设置下载线程数
4. 点击「下载」按钮开始下载

### Pip包管理

1. 在「Pip管理」选项卡中输入包名
2. 点击「安装包」、「卸载包」或「升级包」按钮进行操作
3. 查看操作状态和进度

### 镜像测试

1. 在「设置」菜单中找到镜像测试选项
2. 选择要测试的镜像类型（Python或Pip）
3. 测试完成后可以设置最佳镜像

## 目录结构

- `pyquick.py`：主程序文件
- `run.py`：简单的启动脚本
- `mirror_test.py`：镜像测试模块
- `lang.py`：语言本地化模块
- `log.py`：日志模块
- `utils/`：工具函数
- `modules/`：功能模块
- `pip/`：pip管理相关功能
- `ab/`：关于和过期提示相关
- `config/`：配置文件目录
