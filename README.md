## 使用说明

PyQuick提供以下主要功能：

1. **Python安装包下载**
   - 支持多线程下载
   - 自动获取Python官方最新版本列表
   - 支持暂停/恢复/取消下载

2. **PIP包管理**
   - 查看已安装的包
   - 安装/卸载/升级包
   - 升级pip本身

3. **设置管理**
   - 主题设置
   - 代理设置
   - 日志设置

## 功能特性

- 下载并安装各种版本的Python
- 管理已安装的Python版本
- 提供pip包管理功能
- 简单易用的用户界面
- 支持代理设置
- 自定义主题和界面

## 设置系统

PyQuick提供了一个完整的设置系统，用于管理应用程序配置和自定义UI外观。

### 主要组件

- **设置管理器** (`settings_manager.py`): 负责加载、保存和管理应用程序设置
- **主题管理器** (`theme_manager.py`): 管理应用程序主题，包括默认主题和自定义主题
- **设置UI** (`settings_ui.py`): 提供用户友好的设置界面

### 设置分类

设置面板分为以下几个类别：

- **常规**: 基本设置和应用程序行为
- **界面**: 自定义应用程序外观和界面行为
- **下载**: 下载设置和文件保存选项
- **Python**: Python版本和包管理设置
- **网络**: 网络代理和连接设置
- **高级**: 高级选项和调试设置

### 主题系统

PyQuick支持丰富的主题系统，提供：

- 默认浅色主题
- 默认深色主题
- 自定义主题创建和管理

主题可以完全自定义UI的外观，包括颜色、字体、按钮样式等。
